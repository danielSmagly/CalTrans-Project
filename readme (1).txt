This will save as a zip file. Right click it and open it with java. If you don't have
java, install it from https://java.com/en/download/help/windows_manual_download.html

Before running this, make sure to make a new folder onto your desktop by right 
clicking it and naming it FinishedPDF.
After your pdf is finalized, you can run the pdfprotector.jar by double clicking it.

A dialog box should pop up asking you where your finished pdf is. Navigate through
the file explorer and click your pdf.

Another dialog box will pop up. This is where you will save the pdf in the folder
you just made. You can rename the pdf once this process has ended.

Two more dialog boxes will pop up. Follow the instructions and save your passwords. If
you forget the password, just re-run the program and re-protect the pdf.
